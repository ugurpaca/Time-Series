Three analysts. Three unsolved problems. ARIMA couldn't help.

**Tom** (volatility trader): "I need to predict risk, not just returns. ARIMA models the mean, but I need the variance."

**Lisa** (economist): "GDP, unemployment, inflation, and interest rates all affect each other. ARIMA treats them separately."

**Alex** (retail): "My data has daily, weekly, AND yearly seasonality—plus holidays. SARIMA with s=365 takes forever."

Part 7 introduces the advanced toolkit:

**1. GARCH (Tom's solution)**
→ Models time-varying volatility
→ σ²(t) = ω + α·ε²(t-1) + β·σ²(t-1)
→ Captures volatility clustering
→ Perfect for financial risk modeling

**2. VAR (Lisa's solution)**
→ Vector AutoRegression
→ Models multiple series TOGETHER
→ Granger causality tests
→ Impulse response analysis
→ Everything affects everything

**3. State Space Models**
→ Handle regime changes
→ Time-varying parameters
→ Missing data? No problem
→ Kalman filtering

**4. Prophet (Alex's solution)**
→ Facebook's forecasting tool
→ Multiple seasonalities automatically
→ Easy holiday modeling
→ Robust to outliers and missing data
→ Fast and production-ready

Key insight: ARIMA is powerful, but not universal.

When ARIMA can't:
✗ Model volatility → Use GARCH
✗ Handle multiple series → Use VAR
✗ Deal with regime changes → Use State Space
✗ Handle complex seasonality → Use Prophet

Decision framework included:
• Single series + volatility → GARCH
• Multiple related series → VAR
• Complex seasonality + holidays → Prophet
• Regime changes → State Space
• Simple patterns → Stick with ARIMA
